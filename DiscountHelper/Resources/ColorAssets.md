# Color Assets Setup

In Xcode, create the following colors in `Assets.xcassets`:

## AccentBlue
- Light: #4A90E2
- Dark:  #5FA3F0

## PageBackground
- Light: #F5F5F8
- Dark:  #111111

## CardBackground
- Light: #FFFFFF
- Dark:  #1E1E1E

## SavedGreen
- Light: #27AE60
- Dark:  #2ECC71

---

## How to create in Xcode:
1. Open `Assets.xcassets`
2. Click `+` → New Color Set
3. Name it (e.g. `AccentBlue`)
4. In the attributes inspector, set "Appearances" to "Any, Dark"
5. Set hex values above for each appearance
6. Repeat for all 4 colors
